## Features

1. Weather Forecast:
   Day Forecast: Users can view weather forecasts for the next days.
   City and Country Search: Users can search for weather data by entering the name of a city or country.
   Dynamic Updates: Weather information is dynamically updated based on user input.
   Detailed Weather Data: Provides comprehensive weather details such as temperature, humidity, wind speed, and weather condition descriptions.

2. Responsive Design:
   Mobile Compatibility: The application is designed to be responsive, ensuring a seamless experience across various devices, including desktops, tablets, and smartphones.
   User-Friendly Interface: Intuitive design elements enhance user experience and accessibility.

3. API Integration:
   API Consumption: Utilizes a weather API to fetch real-time weather data.
   Data Visualization: Displays weather information in a visually appealing and easy-to-understand format.

4. Error Handling:
   Input Validation: Validates user input to ensure accurate search results.
   Error Handling: Provides informative error messages in case of invalid inputs or API errors.

### Technologies Used

HTML5: Provides the structure for the web application.
CSS3: Styles the application for an aesthetically pleasing appearance.
JavaScript: Implements dynamic functionality, including API integration and user interaction.
Bootstrap: Utilized for responsive design and layout.
Weather API: Fetches real-time weather data for display.
